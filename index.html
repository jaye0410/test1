Gitでローカルリポジトリを作成するメモ

HomebrewでGitをインストールする

$ brew install git

アカウントをコンフィグに登録する

$ git config --global user.name "jayeXXXX"
$ git config --blobal user.email "xxxx@gmail.com"

ローカルGitリポジトリ用のディレクトリを作成する

$ mkdir gitrepos
$ cd gitrepos
$ mkdir test1
$ cd test1

管理対象のファイル(index.html)を作成する

$ vim index.html

ローカルリポジトリにコミットする手前のindex領域にファイルをステージングする

$ git add index.html

* 全てのファイルをステージングするには以下のコマンドを使う

$ git add --all

状態を確認する

$ git status
On branch master

Initial commit

Changes to be committed:
  (use "git rm --cached ..." to unstage)

	new file:   index.html

./css/style.cssを作成して再度状態を確認する

$ mkdir css
$ cd css
$ vim style.css

$ git status

ローカルリポジトリへコミットする

$ git commit

"Text Editorを開かずコメントを付けたい場合は以下"
$ git commit -m "something to describe."
