Gitでローカルリポジトリを作成するメモ

HomebrewでGitをインストールする

$ brew install git
conflict1

アカウントをコンフィグに登録する

$ git config --global user.name "jayeXXXX"
$ git config --blobal user.email "xxxx@gmail.com"

ローカルGitリポジトリ用のディレクトリを作成する

$ mkdir gitrepos
$ cd gitrepos
$ mkdir test1
$ cd test1

管理対象のファイル(index.html)、作業ディレクトリを作成

$ vim index.html
$ git init

ローカルリポジトリにコミットする手前のindex領域にファイルをステージングする

$ git add index.html
  • *全てのファイルをステージングする

    $ git add --all

  • *カレントディレクトリ配下を全てステージングする

    s$1 git add .

  • *各ファイルの差分を確認してステージングを決める

    $git add -p

* 全てのファイルをステージングするには以下のコマンドを使う

$ git add --all

状態を確認する

$ git status
On branch master

Initial commit

Changes to be committed:
  (use "git rm --cached ..." to unstage)

	new file:   index.html

./css/style.cssを作成して再度状態を確認する

$ mkdir css
$ cd css
$ vim style.css

$ git status

ローカルリポジトリへコミットする

$ git commit

"Text Editorを開かずコメントを付けたい場合は以下"
$ git commit -m "something to describe."

ログを確認する

$ git log

"ファイル名をステータスも確認する場合は以下"
$ git log --name-status
